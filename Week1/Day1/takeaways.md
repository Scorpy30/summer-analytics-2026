# Day 1 Takeaways

* Learned why Python is widely adopted across software, automation, and Machine Learning.
* Understood how Python executes code internally using Interpreter → Bytecode → Python Virtual Machine (PVM).
* `Python Translates & Executes code __ON THE FLY__.`
* Learned the difference between Interactive Mode (REPL) and Script Mode.
* Learned why adding Python to PATH improves terminal workflow.
* Understood the purpose of PEP and why coding standards matter.
* Learned the role of linting in detecting formatting issues and potential bugs early.
* Learned productivity shortcuts in VS Code to reduce repetitive actions.
* Understood how variables reference values instead of storing them conceptually as containers.
* Learned string operations including indexing, slicing, formatting, and common string methods.
* Learned arithmetic operators and their exact use cases (`+`, `-`, `*`, `/`, `//`, `%`, `**`).
* Learned comparison operators and decision-making using conditional statements.
* Understood logical operators (`and`, `or`, `not`) and short-circuit evaluation.
* Learned Python type conversion and Boolean evaluation concepts.
* Explored Python implementations such as CPython, Jython, IronPython and PyPy.
* Learned ideas of Ternary Operators and Chaining Comparison Operators.
* Explored concept of Short-circuit Evalutation.
* Learned that Machine Learning starts with problem definition and data preparation before model training.
* Built the programming and computational foundation required for upcoming analytics modules.
