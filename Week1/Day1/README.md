# Week 1 - Day 1

## Objective

Build a foundational understanding of Python, how Python executes code internally, and get introduced to the basic Machine Learning workflow.

## Topics Covered

* Introduction to Python
* Why Python is widely used
* Python Interpreter
* Bytecode and Python Virtual Machine (PVM)
* REPL vs Script execution
* Variables and Data Types
* Operators and Conditional Statements
* Python implementations (CPython, PyPy)
* Introduction to Machine Learning

## Practical Activities

* Explored how Python code gets executed
* Practiced running Python interactively and through scripts
* Worked with variables and basic data types
* Used conditional logic (`if`, `elif`, `else`)
* Understood the high-level ML development pipeline

## Key Learnings

* Python prioritizes readability and productivity
* Python code is converted into bytecode before execution
* Variables store references to values
* Clean coding practices improve maintainability
* Machine Learning starts with defining the problem before training models
