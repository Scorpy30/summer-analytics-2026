# Week 1 - Day 2

## Objective

Learn Python iteration concepts and build a strong foundation in NumPy by understanding how arrays are stored, manipulated, and optimized for numerical computation.

## Topics Covered

* Iterables and Iteration
* `range()` object
* Loops and traversal
* Functions
* Python memory model
* Python Lists vs NumPy Arrays
* NumPy array creation
* Array dimensions and shapes
* Indexing and Slicing
* Broadcasting
* Views vs Copies
* NumPy performance advantages

## Practical Activities

* Created and manipulated NumPy arrays
* Practiced indexing and slicing operations
* Compared Python lists with NumPy arrays
* Explored array shapes and dimensions
* Built custom matrix patterns
* Investigated memory usage and storage concepts

## Key Learnings

* NumPy achieves speed through contiguous memory storage.
* Python lists store references, not raw values.
* Arrays support efficient vectorized operations.
* Slicing behavior differs between Python lists and NumPy arrays.
* Understanding memory layout is critical for performance.
