# Week 1 - Day 3

## Objective

Learn core Pandas operations for working with tabular data while developing intuition for filtering, regex, missing values, transformations, ranking, and dataframe manipulation.

## Topics Covered

* Pandas DataFrames
* Boolean Filtering
* `.contains()` and Regex
* `.startswith()`
* Missing Values (`NaN`)
* `.drop()` vs assignment
* `.fillna()`
* `.isna()`
* `.apply()`
* `.map()`
* `.where()`
* Adding Columns
* Ranking
* Sorting
* Aggregation
* Joins
* Datetime Conversion

## Practical Activities

* Filtered rows using boolean masks
* Solved regex-based dataframe problems
* Investigated pandas warnings
* Added new columns using multiple approaches
* Ranked values and sorted results
* Worked with missing data strategies
* Compared regex and non-regex solutions

## Key Learnings

* DataFrames accept boolean masks for filtering.
* Pandas string methods internally operate row-wise.
* Missing values require deliberate handling.
* Multiple solutions may produce identical outputs with different tradeoffs.
* Pandas often returns new objects instead of mutating originals.
